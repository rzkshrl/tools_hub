Gambaran Umum & Alur Kerja

Aplikasi ini digunakan untuk mengubah data penjualan dari berbagai satuan (seperti Rtg, DsK, Pcs, Renceng) menjadi satu satuan seragam yaitu Dus. Proses dilakukan langsung di browser Anda — tidak perlu internet dan data tidak dikirim ke server mana pun.

💡
Apa yang dimaksud "konversi ke Dus"?
Misalnya ada 8 Renceng (Rtg) produk BMX Isi 80 Pcs. Karena 1 Dus berisi 8 Renceng, maka 8 Rtg = 1 Dus. Aplikasi ini menghitung otomatis untuk semua produk sekaligus.
✅
Persyaratan file: File Excel Anda harus memiliki minimal 3 kolom: Nama Barang, Satuan, dan Kuantitas. Kolom lain (kode barang, harga, region, dll) akan ikut disimpan di hasil download.

Langkah 2 — Pemetaan Kolom (Wajib Diperiksa)

Setelah file dimuat, aplikasi perlu tahu kolom mana yang berisi Nama Barang, Satuan, dan Kuantitas. Aplikasi akan mencoba menebak otomatis, tapi Anda perlu memastikan pemetaan sudah benar.

Field Keterangan Wajib? Contoh Nama Kolom di Excel
Nama Barang Kolom berisi nama produk Wajib Nama Barang, Item, Produk, Barang
Satuan Kolom berisi kode satuan (Dus, Rtg, DsK, dst) Wajib Satuan, Unit, UOM, Ukuran
Kuantitas Kolom berisi jumlah/angka kuantitas Wajib Kuantitas, Qty, Sum of Kuantitas, Jumlah
Kode Barang Kode/SKU produk (untuk referensi) Opsional Kode Barang, SKU, Kode, FG-
Nilai / Harga Kolom nilai rupiah transaksi Opsional Jumlah, Amount, Sum of Jumlah, Harga
✅
Tip: Jika nama kolom di Excel Anda berbeda (misal "Sum of Kuantitas" bukan "Kuantitas"), aplikasi tetap bisa menebaknya. Namun pastikan dropdown sudah menunjukkan kolom yang tepat sebelum melanjutkan.
💡
Semua kolom asli tetap disimpan: Kolom lain seperti Nama Pelanggan, Region, Tanggal, PIC, dll tidak perlu dipilih di sini — semuanya akan otomatis ikut tersimpan saat Anda download hasil.

Langkah 3 — Aturan Konversi (Konfigurasi Produk)

Aturan konversi menentukan berapa satuan lain setara 1 Dus untuk setiap jenis produk. Sudah tersedia aturan default, tapi Anda bisa menambah atau mengubah sesuai kebutuhan.

💾
Aturan tersimpan otomatis di browser. Setiap perubahan langsung disimpan — Anda tidak perlu klik "Simpan" secara manual. Aturan akan tetap ada meski browser ditutup dan dibuka kembali.
📋
Aturan Default yang Sudah Ada
Aplikasi sudah memiliki aturan untuk produk BMX, Heppi Twist, dan Grandchoco. Lihat tabel referensi di bagian 9 untuk rincian konversi setiap satuan.
🔑
Cara Kerja "Keyword"
Setiap aturan memiliki keyword — kata kunci yang dicocokkan ke nama barang. Misalnya keyword isi 80 akan mencocokkan semua produk yang namanya mengandung "isi 80" seperti "BMX Chocoball Isi 80 Pcs" atau "DO RE MI Heppi Twist Vanila Isi 80 Pcs".

Jika produk punya beberapa keyword, pisahkan dengan koma: isi 80, 80 pcs
➕
Tambah Aturan Baru
Klik tombol + Tambah Aturan Konversi di bawah daftar aturan. Isi:
  1. Nama Aturan — deskripsi singkat untuk mudah dikenali
  2. Keyword — kata dari nama barang yang menjadi penanda
  3. Satuan dan jumlah per Dus — misal: Rtg = 8 per Dus, Pcs = 80 per Dus
↩️
Reset ke Default
Jika aturan tidak sengaja terhapus atau berubah, klik tombol ↺ Default di header kartu Aturan Konversi untuk mengembalikan ke aturan bawaan.
💾
Ekspor & Impor Aturan
Anda bisa menyimpan aturan ke file JSON untuk dibagikan ke rekan kerja atau digunakan di komputer lain:
  • Klik 💾 untuk ekspor (download file aturan.json)
  • Klik 📥 untuk impor (upload file aturan.json dari komputer)
⚠️
Penting — urutan pencocokan keyword: Aturan dicocokkan dari urutan atas ke bawah. Jika ada produk yang cocok di lebih dari satu aturan, aturan yang paling atas yang digunakan. Susun aturan dari yang paling spesifik di atas ke yang paling umum di bawah.

Tab Pivot — Laporan Tabel Silang

Pivot adalah fitur untuk membuat laporan tabel silang — misalnya Total Dus per Produk berdasarkan Region, atau per Brand berdasarkan Bulan. Berguna untuk analisis distribusi penjualan.

🔢
Cara Membuat Pivot
  1. Klik tab 🔢 Pivot
  2. Pilih Dimensi Baris — misal: Nama Barang
  3. Pilih Dimensi Kolom — misal: REGION (opsional)
  4. Pilih Nilai — Total Dus, Qty Asli, atau Jumlah Baris
  5. Tabel pivot langsung muncul otomatis
✅
Contoh hasil pivot: Dimensi Baris = Nama Barang, Dimensi Kolom = REGION → Anda mendapat tabel yang menunjukkan berapa Dus setiap produk terjual di setiap region. Baris terakhir menampilkan Grand Total otomatis.
⬇️
Download Pivot
Klik tombol ⬇ Download Pivot untuk mengunduh tabel pivot ke file Excel tersendiri. Ini berguna untuk laporan atau dipresentasikan.

Pertanyaan Umum (FAQ)

Apakah data saya aman? Apakah dikirim ke internet?
Tidak. Semua proses terjadi di browser komputer Anda. Data tidak dikirim ke server mana pun. Aplikasi ini bisa digunakan bahkan tanpa koneksi internet (mode offline), asalkan file HTML-nya sudah dibuka sebelumnya.
Ada produk yang statusnya "⚠ Tidak ada aturan" meskipun sudah ada aturannya. Kenapa?
Kemungkinan keyword tidak cocok. Pastikan keyword di aturan ada di dalam nama barang di data. Contoh: keyword "isi 80" harus muncul sebagai bagian dari nama barang seperti "BMX Chocoball Isi 80 Pcs". Pencocokan tidak case-sensitive tapi harus persis. Cek juga apakah ada spasi ekstra atau typo.
Aturan yang saya tambahkan tiba-tiba hilang. Kenapa?
Aturan disimpan di localStorage browser. Jika Anda menggunakan mode incognito/private, membersihkan data browser, atau menggunakan browser berbeda, aturan akan hilang. Solusinya: gunakan tombol 💾 Ekspor untuk menyimpan aturan ke file JSON, lalu 📥 Impor saat dibutuhkan.
Bisa tidak digunakan di HP/smartphone?
Bisa, tampilan akan menyesuaikan layar (responsive). Namun pengalaman terbaik adalah di komputer dengan layar lebar karena tabel dan panel sidebar lebih mudah dilihat.
File saya berukuran besar (ribuan baris). Apakah bisa diproses?
Bisa. Aplikasi diuji untuk file dengan ribuan baris. Proses konversi dilakukan di background sehingga browser tidak membeku. Tabel hasil menampilkan maksimal 50 baris per halaman secara default (bisa diubah ke 500), namun download Excel akan mengandung semua data.
Kenapa ada baris negatif (angka merah) di hasil?
Angka negatif adalah transaksi retur (barang kembali). Ini normal dan sengaja ditampilkan merah agar mudah diidentifikasi. Gunakan toggle Retur untuk menampilkan hanya transaksi retur, atau abaikan jika tidak relevan.
Satuan yang ada di data saya tidak ada di aturan default. Apa yang harus dilakukan?
Tambahkan baris satuan baru di aturan yang sudah ada, atau buat aturan baru. Misalnya jika ada satuan "Karton" yang setara DsK (6 per Dus untuk produk 6x24), tambahkan baris: Satuan = "Karton", Per Dus = 6 di aturan Grandchoco.
Apakah aplikasi ini bisa digunakan untuk produk selain BMX dan Grandchoco?
Ya. Tambahkan aturan baru dengan keyword dari nama produk Anda dan masukkan rasio konversi yang sesuai. Tidak ada batasan jumlah aturan yang bisa ditambahkan.

Panduan Visual — Walkthrough Lengkap

Ikuti langkah-langkah di bawah ini secara berurutan dari awal hingga mendapatkan file hasil konversi. Setiap langkah dilengkapi penjelasan apa yang harus dilihat dan dilakukan.

Buka file aplikasi di browser
Cari file konversi-satuan-dus.html di komputer Anda (biasanya di folder Downloads). Klik dua kali untuk membukanya di browser. Anda akan melihat tampilan dengan header hijau bertuliskan "Konversi Satuan → Dus" dan dua panel: panel kiri (pengaturan) dan panel kanan (hasil).

Tidak perlu koneksi internet. Tidak perlu install apapun.
✅ Yang terlihat: Halaman terbuka dengan area upload di panel kiri dan area kosong bertuliskan "Upload file Excel" di panel kanan.
Upload file Excel data penjualan
Di panel kiri, klik area bertekstur garis putus-putus bertuliskan "Klik atau drag & drop file Excel". Cari dan pilih file Excel Anda (misal: CLOSING_FIX.xlsx). Tunggu beberapa detik.
✅ Yang terlihat: Bar hijau muncul di bawah area upload menampilkan nama file dan ukuran. Dropdown "Pilih Sheet" dan label "Satuan Terdeteksi" muncul otomatis.
Pilih sheet yang benar
Jika muncul dropdown "Sheet", pilih sheet yang berisi data penjualan. Biasanya bernama Sheet1 atau sesuai nama bulan/periode. Di bawah dropdown akan tertera jumlah baris dan kolom yang ditemukan (misal: "2.695 baris · 19 kolom").
✅ Yang terlihat: Informasi jumlah baris muncul, dan satuan-satuan yang terdeteksi tampil sebagai chip biru seperti Dus Rtg DsK
Periksa pemetaan kolom
Kartu "🗂️ Pemetaan Kolom" muncul di bawah upload. Periksa bahwa setiap dropdown menunjukkan kolom yang benar:
  • Nama Barang → kolom berisi nama produk (misal: "Nama Barang")
  • Satuan → kolom berisi kode satuan (misal: "Satuan")
  • Kuantitas → kolom berisi angka jumlah (misal: "Sum of Kuantitas")
Jika ada yang salah, klik dropdown tersebut dan pilih kolom yang tepat.
✅ Yang terlihat: Ketiga dropdown wajib sudah menunjukkan kolom yang benar. Tombol "🔄 Proses Konversi" sudah aktif (tidak abu-abu).
Cek aturan konversi (opsional tapi disarankan)
Gulir ke bawah ke kartu "⚙️ Aturan Konversi". Pastikan aturan untuk produk Anda sudah ada. Untuk data BMX/Grandchoco, aturan default sudah cukup. Jika ada produk lain, tambahkan aturan sekarang sebelum memproses.
✅ Yang terlihat: Minimal 3-4 blok aturan sudah tersedia dengan keyword dan rasio konversi masing-masing.
Klik "🔄 Proses Konversi"
Klik tombol hijau besar "🔄 Proses Konversi" di bagian bawah panel kiri. Tunggu sebentar (proses biasanya kurang dari 3 detik untuk ribuan baris).
✅ Yang terlihat: Notifikasi hijau muncul di pojok bawah kanan (misal: "Selesai! 2.680/2.695 baris berhasil"). Panel kanan menampilkan 4 angka statistik dan tabel hasil.
Periksa tab "⚠ Perlu Cek"
Klik tab "⚠ Perlu Cek" di panel kanan. Jika ada angka di dalam tanda kurung (misal "⚠ Perlu Cek (15)"), artinya ada baris yang gagal dikonversi. Untuk setiap produk yang muncul, klik tombol + Aturan dan isi form aturannya.

Jika tab menunjukkan "Perlu Cek (0)" atau muncul pesan "Sempurna! Semua baris berhasil", lanjut ke langkah berikutnya.
✅ Yang terlihat: Setelah menambah aturan dan memproses ulang, angka di tab "Perlu Cek" berkurang atau menjadi 0.
Analisis hasil di berbagai tab (opsional)
Eksplorasi hasil sebelum mendownload:
  • Tab 📈 Grafik — lihat visualisasi total Dus per produk
  • Tab 📦 Per Produk — rekap total Dus per nama barang
  • Tab 🔢 Pivot — buat tabel silang Produk × Region
  • Filter chip di atas tabel — filter berdasarkan Region, Brand, atau Bulan
✅ Yang terlihat: Grafik batang horizontal menampilkan produk dengan total Dus terbesar di urutan teratas.
Download hasil Excel
Klik tombol ⬇ Download Excel di pojok kanan atas tabel. File Excel dengan 4 sheet akan tersimpan otomatis ke folder Downloads Anda.
✅ Yang terlihat: Notifikasi "File Excel (4 sheet) berhasil didownload!" muncul. File bernama namafile_konversi_dus.xlsx muncul di folder Downloads.

16 Rtg ÷ 4 = 4 Dus. Status berubah jadi ✓ OK.

Kasus 4 — Laporan Penjualan per Region
Ingin tahu berapa Dus yang terjual di setiap wilayah/region
Setelah konversi selesai, ada dua cara untuk melihat ini:
Cara A — Gunakan Filter Chip
Di atas tabel, klik chip filter REGION → pilih wilayah yang ingin dilihat. Lalu aktifkan toggle Grouped untuk melihat total per produk di region tersebut.
Cara B — Gunakan Tab Pivot
Klik tab 🔢 Pivot → pilih Dimensi Baris: "Nama Barang" → Dimensi Kolom: "REGION" → Nilai: "Total Dus". Hasilnya adalah tabel yang menampilkan semua produk vs semua region sekaligus, dengan Grand Total di baris paling bawah.
💡 Tips: Klik tombol ⬇ Download Pivot untuk menyimpan tabel silang ini ke Excel secara terpisah — berguna untuk presentasi atau laporan bulanan.

Tips & Trik Pengguna Mahir

⚡
Keyword yang lebih spesifik = lebih akurat
Gunakan keyword yang unik dan tidak muncul di nama produk lain. Misalnya "isi 80 pcs" lebih spesifik daripada hanya "80". Pisahkan dengan koma jika ada variasi penulisan: isi 80, 80 pcs
📤
Backup aturan sebelum ganti file Excel
Klik 💾 Ekspor setiap selesai mengatur aturan. Simpan file aturan_konversi_dus.json di folder yang aman. Jika pindah komputer atau bersihkan cache browser, impor kembali dengan 📥 Impor.
🔍
Hover pada kolom Qty (Dus) untuk detail
Arahkan kursor ke angka di kolom Qty (Dus). Akan muncul tooltip yang menjelaskan perhitungan lengkap, misal: "16 Rtg ÷ 4 = 4,0000 Dus [aturan: Produk Isi 40 Pcs]"
📋
Salin ke Google Sheets tanpa download
Klik 📋 Salin, buka Google Sheets, lalu tekan Ctrl+V. Data tabel (sesuai filter aktif) langsung terpaste rapi dalam format tab-separated yang Sheets kenali sebagai kolom.
🎯
Filter + Grouped untuk rekap cepat
Klik filter chip Region → pilih satu wilayah → aktifkan toggle Grouped. Hasilnya: total Dus per produk khusus untuk wilayah itu. Cocok untuk laporan per area sales.
🔄
Proses ulang tidak reset aturan
Anda bisa klik "🔄 Proses Konversi" berkali-kali tanpa takut aturan hilang atau data rusak. Setiap klik memproses ulang seluruh data dengan aturan terkini.
📊
Ubah grafik menjadi "Top 20" untuk file besar
Jika produk sangat banyak, grafik bisa terlihat sesak. Ubah dropdown dari "Top 10" ke "Top 20" — atau gunakan tab "Per Produk" dengan fungsi pencarian yang lebih fleksibel.
🖨️
Cetak panduan ini sebagai referensi
Tekan Ctrl+P (atau Cmd+P di Mac) untuk mencetak panduan ini. Layout dioptimalkan untuk cetak — navigasi sticky tidak akan tercetak.
💡
Scoping keyword untuk produk yang namanya mirip: Jika ada "BMX Chocoball Isi 80 Pcs" dan "BMX Big Chocoball Isi 40 Pcs", pastikan keyword cukup spesifik. Gunakan chocoball isi 80 dan big chocoball isi 40 — bukan hanya chocoball yang akan mencocokkan keduanya ke aturan pertama.

Panduan Mengatasi Masalah

❌
File Excel tidak bisa diupload / tidak muncul apa-apa setelah dipilih
🔎 Kemungkinan penyebab: Format file tidak didukung, file rusak, atau ukuran file terlalu besar untuk browser.
Solusi:
  1. Pastikan file berekstensi .xlsx, .xls, atau .csv. File .xlsm (dengan macro) mungkin bermasalah — simpan ulang sebagai .xlsx dulu.
  2. Coba buka file di Excel terlebih dahulu untuk memastikan tidak rusak.
  3. Untuk file sangat besar (>50 MB), coba filter/kurangi data di Excel dulu sebelum upload.
  4. Coba browser berbeda (Chrome atau Edge disarankan).
❌
Dropdown pemetaan kolom tidak mendeteksi kolom yang benar
🔎 Kemungkinan penyebab: Nama kolom di Excel tidak mengandung kata kunci yang dikenali (misal: "Qty" padahal aplikasi menebak dari kata "Kuantitas").
Solusi:
  1. Klik dropdown yang salah dan pilih kolom yang tepat secara manual.
  2. Alternatif: ubah nama kolom di Excel agar lebih standar (misal: "Qty" → "Kuantitas"), lalu upload ulang.
  3. Tidak perlu mengubah nama kolom — cukup pilih manual dari dropdown, aplikasi akan mengingat pilihan Anda.
❌
Semua atau banyak baris statusnya "⚠ Tidak ada aturan" meski aturan sudah diisi
🔎 Kemungkinan penyebab: Keyword tidak cocok dengan isi kolom Nama Barang, atau kolom Nama Barang salah dipetakan.
Solusi:
  1. Klik 👁 Preview dan lihat isi kolom Nama Barang — pastikan mengandung teks yang sama dengan keyword Anda.
  2. Pastikan kolom Nama Barang sudah dipetakan ke kolom yang benar (bukan ke kolom kode barang atau region).
  3. Cek keyword: jika nama barang di data adalah "BMX Chocoball Isi 80 Pcs", keyword "isi 80" akan cocok. Tapi keyword "Isi80" (tanpa spasi) tidak akan cocok.
  4. Coba keyword yang lebih pendek dan pasti ada di nama barang, misal hanya "chocoball" atau "grandchoco".
❌
Status "✕ Satuan tidak dikenal" meski aturan untuk produk itu sudah ada
🔎 Kemungkinan penyebab: Satuan di data (misal "RTG" huruf kapital semua) tidak sama persis dengan satuan di aturan (misal "Rtg").
Solusi:
  1. Pencocokan satuan tidak case-sensitive — "RTG", "rtg", "Rtg" seharusnya semua cocok. Jika masih gagal, cek apakah ada spasi tersembunyi (misal "Rtg " dengan spasi di akhir).
  2. Klik Preview dan lihat isi kolom Satuan untuk mengetahui nilai pastinya.
  3. Di aturan konversi, tambahkan baris satuan baru dengan nilai persis seperti yang ada di data (salin-paste jika perlu).
  4. Lihat chip satuan terdeteksi (di bawah area upload) — pastikan semua satuan yang ada di sana sudah didaftarkan di aturan yang sesuai.
❌
Hasil konversi tampak salah — angka Dus terlalu besar atau terlalu kecil
🔎 Kemungkinan penyebab: Nilai "Per Dus" di aturan terbalik (misal diisi 80 padahal seharusnya 8).
Solusi:
  1. Hover ke angka di kolom Qty (Dus) untuk melihat detail perhitungan: misal "16 Rtg ÷ 8 = 2 Dus". Cek apakah pembagi sudah benar.
  2. Rumus yang digunakan: Qty (Dus) = Qty Asli ÷ Nilai Per Dus. Jika 1 Dus = 8 Rtg, maka nilai Per Dus untuk satuan Rtg adalah 8.
  3. Perbaiki nilai Per Dus di aturan → klik "🔄 Proses Konversi" ulang.
❌
Tombol "Download Excel" tidak berfungsi / tidak ada file yang tersimpan
🔎 Kemungkinan penyebab: Popup download diblokir browser, atau belum ada data hasil konversi.
Solusi:
  1. Pastikan sudah klik "🔄 Proses Konversi" terlebih dahulu — tombol Download baru aktif setelah ada hasil.
  2. Cek apakah browser memblokir download. Lihat pojok kanan atas browser — biasanya ada ikon atau notifikasi "Download diblokir". Izinkan download dari aplikasi ini.
  3. Cek folder Downloads di komputer Anda — file mungkin sudah tersimpan tanpa notifikasi.
  4. Coba browser Chrome atau Edge yang paling kompatibel.
❌
Filter dinamis (chip Region/Brand/dll) tidak muncul setelah proses
🔎 Kemungkinan penyebab: Kolom kategorikal di data memiliki terlalu banyak nilai unik (>80), atau kolom tersebut dikenali sebagai kolom numerik.
Solusi:
  1. Filter otomatis hanya muncul untuk kolom dengan maksimal 80 nilai unik. Jika kolom Region misalnya memiliki 100+ nilai berbeda (karena nama yang tidak konsisten), filter tidak muncul.
  2. Bersihkan data di Excel dulu: standarisasi nama region/brand yang tidak konsisten (misal: "Jawa Barat" vs "jabar" vs "JABAR" → pilih satu format).
  3. Sebagai alternatif, gunakan kolom pencarian di tabel untuk mencari secara manual.
❌
Halaman terasa lambat atau browser tidak merespons saat memproses
🔎 Kemungkinan penyebab: File Excel memiliki banyak baris (lebih dari 50.000 baris) sehingga membutuhkan waktu lebih lama.
Solusi:
  1. Tunggu beberapa saat — proses berjalan di background. Browser mungkin menampilkan pesan "halaman tidak merespons" — klik "Tunggu" (bukan "Keluar").
  2. Untuk file sangat besar, bagi menjadi beberapa file per bulan atau per wilayah, proses secara terpisah, lalu gabungkan di Excel.
  3. Tutup tab browser lain yang tidak diperlukan untuk membebaskan memori.

Glosarium Istilah

Daftar istilah teknis yang digunakan di aplikasi dan panduan ini:

Dus
Satuan tujuan konversi. Satu Dus adalah satu kardus/karton besar yang berisi produk lengkap sesuai standar pabrik.
Rtg / Renceng
Satuan berupa strip/renceng produk. Satu renceng biasanya berisi 10 Pcs, dan beberapa renceng membentuk 1 Dus.
DsK / Duplex
Satuan karton kecil/duplex yang merupakan pembungkus lebih kecil dari Dus. Beberapa DsK membentuk 1 Dus.
Pcs
Satuan satuan terkecil — piece atau butir produk individual. Dikonversi ke Dus dengan membagi jumlah Pcs per Dus.
Keyword
Kata kunci yang digunakan untuk mencocokkan nama barang ke aturan konversi. Satu aturan bisa memiliki banyak keyword dipisah koma.
Aturan Konversi
Konfigurasi yang mendefinisikan: untuk produk dengan keyword tertentu, berapa satuan X setara 1 Dus. Disimpan di browser secara lokal.
Pemetaan Kolom
Proses menunjukkan ke aplikasi kolom mana di Excel yang berisi Nama Barang, Satuan, dan Kuantitas.
Status OK
Baris berhasil dikonversi: aturan ditemukan dan satuan dikenali. Kolom Qty (Dus) sudah terisi dengan nilai.
Status Perlu Cek
Baris gagal dikonversi. Bisa karena tidak ada aturan yang cocok, atau satuan tidak terdaftar di aturan yang cocok.
Mode Grouped
Tampilan di mana semua baris dengan nama produk yang sama digabungkan menjadi satu baris, dengan total Qty Dus dijumlahkan.
Retur
Transaksi dengan kuantitas negatif, menandakan barang dikembalikan dari pelanggan ke gudang. Ditampilkan merah di tabel.
Pivot
Tabel silang yang merangkum data berdasarkan dua dimensi sekaligus — misal Produk (baris) × Region (kolom) = Total Dus.
Filter Dinamis
Chip filter yang muncul otomatis berdasarkan kolom kategorikal di data Anda (Region, Brand, Bulan, PIC, dll).
localStorage
Mekanisme penyimpanan bawaan browser yang digunakan untuk menyimpan aturan konversi. Data tidak dikirim ke internet — tersimpan hanya di komputer Anda.
Per Dus
Nilai di aturan konversi yang menunjukkan berapa unit satuan X dibutuhkan untuk membentuk 1 Dus. Rumus: Qty Dus = Qty Asli ÷ Per Dus.
Sheet
Lembar kerja di dalam satu file Excel. Satu file bisa memiliki beberapa sheet — aplikasi meminta Anda memilih sheet mana yang berisi data penjualan.
✅
Rumus Dasar Konversi:
Qty (Dus) = Qty Asli ÷ Jumlah [satuan] per Dus
Contoh: 24 Rtg ÷ 8 Rtg/Dus = 3 Dus
Contoh: 160 Pcs ÷ 80 Pcs/Dus = 2 Dus
Contoh: 12 DsK ÷ 6 DsK/Dus = 2 Dus
